# MyMvcApp

A minimal ASP.NET Core MVC app (.NET 8).

## Prerequisites

Install the [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) if you don't already have it. Check with:

```
dotnet --version
```

## Run it

From this folder (the one containing `MyMvcApp.csproj`):

```
dotnet restore
dotnet run
```

You'll see output like:

```
Now listening on: http://localhost:5000
```

Open **http://localhost:5000** in your browser. `launchSettings.json` is set to auto-open your browser when you run via `dotnet run` (or an IDE), so it may open automatically.

## Project layout

- `Program.cs` — app startup and routing
- `Controllers/HomeController.cs` — Home and Privacy actions
- `Views/` — Razor views (`Home/Index.cshtml`, `Home/Privacy.cshtml`, shared layout)
- `wwwroot/` — static CSS/JS (Bootstrap is pulled from a CDN in the layout, no local copy needed)

## Note

This project was hand-written (not generated by `dotnet new mvc`) because the environment that built it had no internet access to Microsoft's SDK servers and couldn't run `dotnet` itself. The file structure mirrors the standard MVC template, so `dotnet run` should work out of the box once you have the SDK installed locally.
